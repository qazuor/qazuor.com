## Color Palette Generator

Generate beautiful, harmonious color palettes using color theory principles.

## Features

- Multiple harmony rules (complementary, triadic, analogous)
- Base color picker
- Automatic palette generation
- Export in different formats (HEX, RGB, HSL)
- Color accessibility checker
- Save favorite palettes
- WCAG contrast ratio validation

## Usage

1. Pick a base color
2. Choose harmony rule
3. Generate palette variations
4. Check accessibility ratings
5. Export in preferred format

Essential tool for designers and developers working with colors.
