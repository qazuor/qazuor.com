# Light Sweep Animation

Create a rotating light beam using conic gradients and custom properties.

```css
@property --angle {
  syntax: "<angle>";
  initial-value: 0deg;
  inherits: false;
}

.card .sweep {
  background: conic-gradient(
    from var(--angle),
    transparent 0%,
    rgba(255, 255, 255, 0.3) 10%,
    transparent 20%
  );
  animation: rotate 3s linear infinite;
}

@keyframes rotate {
  to { --angle: 360deg; }
}
```
