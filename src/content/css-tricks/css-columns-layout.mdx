# CSS Columns

Create responsive multi-column text layouts automatically.

```css
.article {
  columns: 3 300px;
  column-gap: 2rem;
  column-rule: 1px solid #ddd;
}
```

The browser creates up to 3 columns, each at least 300px wide - no media queries needed.
