# CSS corner-shape

Go beyond rounded corners with new shape options.

```css
.card {
  border-radius: 50px;
  corner-shape: squircle; /* iOS-style */
}
```

Available values: `round` (default), `squircle`, `bevel`, `scoop`, `notch`
