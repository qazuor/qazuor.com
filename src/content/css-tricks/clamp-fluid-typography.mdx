# Fluid Sizing with clamp()

Use `clamp(min, preferred, max)` for smooth responsive scaling without breakpoints.

```css
font-size: clamp(1rem, 0.5rem + 2vw, 2.5rem);
```

- **Minimum**: `1rem` - never smaller
- **Preferred**: `0.5rem + 2vw` - scales with viewport
- **Maximum**: `2.5rem` - never larger

## Works With Any Property

```css
padding: clamp(1rem, 5vw, 3rem);
gap: clamp(0.5rem, 2vw, 2rem);
width: clamp(200px, 30vw, 400px);
```
