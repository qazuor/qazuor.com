## Overview

An elegant weather dashboard providing comprehensive weather information with stunning visuals and interactive maps.

## Key Features

- Current weather conditions
- 7-day forecast
- Hourly predictions
- Interactive weather maps
- Location-based alerts
- Multi-city tracking
- Weather radar visualization

## Technical Details

- **Frontend**: React with TypeScript
- **API Integration**: OpenWeather API
- **Maps**: Mapbox GL JS
- **Charts**: Recharts
- **Icons**: Custom weather icons
