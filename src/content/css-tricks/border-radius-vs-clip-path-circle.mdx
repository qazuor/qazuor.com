# Creating Circles

`border-radius: 50%` looks circular but has a square hitbox. `clip-path: circle()` clips the actual click area.

```css
/* Square hitbox */
.avatar {
  border-radius: 50%;
}

/* Circular hitbox */
.avatar {
  clip-path: circle(50%);
}
```
