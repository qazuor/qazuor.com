# Mirror Reflection

Create reflections with a single CSS property (WebKit only).

```css
img {
  -webkit-box-reflect: below 0px linear-gradient(
    to bottom,
    transparent,
    rgba(0, 0, 0, 0.5)
  );
}
```

| Part | Purpose |
|------|---------|
| `below/above/left/right` | Direction |
| `0px` | Gap from original |
| `linear-gradient` | Fade mask |
