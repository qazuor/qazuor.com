# Animate Height with Grid

CSS can't animate `height: auto`, but it can animate `grid-template-rows` from `0fr` to `1fr`.

```css
.accordion {
  display: grid;
  grid-template-rows: 0fr;
  transition: grid-template-rows 0.3s;
}

.accordion > div {
  overflow: hidden;
}

.accordion.open {
  grid-template-rows: 1fr;
}
```
