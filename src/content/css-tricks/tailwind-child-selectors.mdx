# Tailwind Child Selectors

Apply styles to children from the parent element.

```html
<!-- Before: Repetitive -->
<table>
  <td class="px-3 py-2">Cell 1</td>
  <td class="px-3 py-2">Cell 2</td>
  <td class="px-3 py-2">Cell 3</td>
</table>

<!-- After: Clean -->
<table class="[&_td]:px-3 [&_td]:py-2">
  <td>Cell 1</td>
  <td>Cell 2</td>
  <td>Cell 3</td>
</table>
```
