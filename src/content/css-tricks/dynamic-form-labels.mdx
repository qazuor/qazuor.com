# Dynamic Form Labels

Use `:has()` to automatically mark required and optional fields.

```css
label:has(+ input:required)::after {
  content: " *";
  color: red;
}

label:has(+ input:optional)::after {
  content: " (optional)";
  color: #777;
}
```
