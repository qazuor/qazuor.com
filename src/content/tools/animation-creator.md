## CSS Animation Creator

Build complex CSS animations with an intuitive keyframes editor and live
preview.

## Features

- Visual keyframes editor
- Multiple animation properties (transform, opacity, etc.)
- Timing function selector
- Duration and delay controls
- Animation loop options
- Live preview with test element
- Export as CSS @keyframes

## Usage

1. Create keyframes at different percentages
2. Set properties for each keyframe
3. Configure timing and duration
4. Preview animation in real-time
5. Copy generated CSS code

Perfect for creating engaging micro-interactions and smooth transitions.
