# CSS Step Counters

Generate "Step 2 of 5" text automatically without JavaScript.

```css
:root {
  counter-reset: currentStep totalSteps;
}

.step {
  counter-increment: totalSteps;
}

.step:has(~ .step.active),
.step.active {
  counter-increment: totalSteps currentStep;
}

.message::after {
  content: "Step " counter(currentStep) " of " counter(totalSteps);
}
```
