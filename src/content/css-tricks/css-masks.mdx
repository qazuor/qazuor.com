# CSS Masks

Use gradients to control element opacity for fade effects.

```css
.fade-bottom {
  mask-image: linear-gradient(to bottom, black 60%, transparent);
}

.spotlight {
  mask-image: radial-gradient(circle, black 40%, transparent 70%);
}
```
